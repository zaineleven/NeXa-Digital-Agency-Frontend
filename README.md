# NeXa Digital Agency

A modern, fully responsive multi-page digital agency website built using **HTML5, CSS3, and Vanilla JavaScript** as part of the **DecodeLabs Full Stack Development Internship (Project 1 – Responsive Architecture).**

The project focuses on creating a clean, responsive, and interactive user interface without using any frontend frameworks or libraries, emphasizing strong fundamentals in web development.

## Features

* Fully responsive design (Mobile, Tablet & Desktop)
* Multi-page website structure
* Modern landing page with interactive UI
* Light/Dark theme toggle with local storage persistence
* Scroll reveal animations using Intersection Observer API
* Custom animated background
* Infinite scrolling services marquee
* Responsive navigation bar
* Contact form UI
* Semantic HTML5 structure
* Accessibility-friendly implementation
* Clean and reusable code structure

## Installation

Clone the repository

```bash
git clone https://github.com/zaineleven/nexa-digital-agency.git
```

Navigate into the project

```bash
cd nexa-digital-agency
```

Open `index.html` in your browser.

No additional installation or dependencies are required.
